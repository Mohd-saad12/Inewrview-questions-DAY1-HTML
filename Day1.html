Q.1-Difference between Div and span?

Div-It is a block level element and it is used to wrap section of any large block of content or headers,footer,mainly use to create divisions
Span-It is an Inline element and it is used to merge row/coloumn or few words witin a paragraph

Q.2-Difference between Class and Id's

Class is used for applying the same styles or behaviours to multiple elements. While ID uniquely identifies a single element on a page . Elements can have multiple classes but only one ID.

Q.3-What are self closing tags?

Self closing tags in HTML are tags that do not require a closing tag. These tags include a forward slash before the closing tag.
for example: br, img, input

Q.5-Difference between Block and Inline elements?

Block Element - It take up the full width of their container and start on a new line.

Inline Element- It take up only as much width as necessary and do not start on a new line.

Q.6-Difference between em and strong?

em- Mark text that should be emphasized, usually resulting in italics text.

Strong- Marks text that should be strongly emphasized,used in bold text .

Q.7-What are attributes ?

Attributes in HTML are additional information provided within an HTML tag to define the properties and behaviors of the element. It have name and Value.
